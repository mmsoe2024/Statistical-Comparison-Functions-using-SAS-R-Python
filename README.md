# Statistical comparison functions using SAS, R & Python
The Statistics Team members within the Surveillance Branch (SB), DHQP, have developed a series of statistical comparison functions.  These functions have been developed to conduct statistical tests for epidemiological measures, consistent with the methods used by NHSN. Statistical tests are based on the mid-p exact method. Measures tested include common epidemiological measures such as incidence rates, proportions, standardized metrics (standardized infection ratio ‘SIR’s, standardized utilization ratio ‘SUR’s, standardized antimicrobial administration ratio ‘SAAR’s and others) and ratios. 

The original language for these comparisons were produced using SAS in macro form in early 2010s.  Six SAS macros were distributed for internal analysis use within SB, utilized within the NHSN web application, as well as published on the NHSN website for public use (https://www.cdc.gov/nhsn/ps-analysis-resources/index.html). In 2026, NHSN updated functionality of these six macros and developed equivalent Python and R versions comparable to the original SAS implementations.

For github internal users (CDCENT), we have provided  a quick reference guide using NHSN SAS macros as an example use case on the landing page. Subfolders for SAS, Python and R have been created to store statistical functions for corresponding programming languages.  We’ve chosen to use the term ‘function’ to more generically refer and describe these routines although they were originally developed as SAS macros at first.
